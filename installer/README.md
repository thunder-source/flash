# Flash Compiler

A high-performance compiler written in pure x86-64 assembly.

## Installation

### Windows Installer

1. Download `FlashCompiler_Setup.exe` from the releases page
2. Run the installer and follow the on-screen instructions
3. Launch the compiler from the Start Menu or Desktop shortcut

### Portable Version

1. Download `FlashCompiler_Portable.zip`
2. Extract to a folder of your choice
3. Run `flash_compiler.exe`

## Building from Source

### Requirements

- NASM (Netwide Assembler)
- Microsoft Visual Studio 2022 with C++ build tools
- Inno Setup (for creating the installer)

### Build Steps

1. Clone the repository
2. Run `installer\build_installer.bat`
3. Find the installer in the `dist` directory
